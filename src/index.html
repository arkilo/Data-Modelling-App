Introduction

Welcome to the data modelling interface. This interface is used to communicate with a backend math library that processes the input data. On the left side you can input a csv file containing numerical data. The data can be processed if it has an underlying exponential function. After that you can choose how must you would like to project the data onwards and the projections will be displayed in red on the graphs.

Usage

Using the module on the left, input a csv file, it will be parsed and the data will be extracted by the backend automatically. You can then choose the number of days upto which you would like to view the projections. After that press process and the data will be projected and displayed on the main graphs.
